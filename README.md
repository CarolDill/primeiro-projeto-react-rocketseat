# Github Explorer

## Stuff being used

- React (CRA)
- Styled-Components
- TypeScript
- Lottie
- Axios
- Polished

## Motivation

The goal is always learn something more, course by Rocketseat.

## How to Run

Install the dependencies and devDependencies by running:

```sh
yarn install
```

Start the dev server:

```sh
yarn start
```
